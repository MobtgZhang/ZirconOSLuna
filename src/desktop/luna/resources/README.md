# ZirconOSLuna Luna 主题资源

本目录下的光栅图由仓库内脚本 **程序化生成**，视觉语言为项目原创：**锆石几何（Zircon）+ 月夜（Luna）+ NT 5.x 式壳层布局**，刻意避免与 Windows / ReactOS 自带图标在构图上可混淆的相似性（无「蓝天丘陵」壁纸、无地球+e 标、无经典纸篓剪影等）。

## 技术规格

| 项目 | 约定 |
|------|------|
| 色彩空间 | sRGB |
| 桌面 / 系统 / 菜单图标 | 32×32 px，PNG，RGBA，透明底 |
| 托盘图标 | 16×16 px |
| 开始按钮贴片 | 108×30 px（与 `theme.zig` 中 `START_BTN_*` 一致） |
| 标题栏按钮条 | 63×21 px，三格从左到右：**最小化 \| 最大化 \| 关闭**，每格 21×21 |
| 光标 | 32×32 px，RGBA；热点见下表 |
| 壁纸 | 1280×720 px，PNG |
| 文件命名 | `snake_case`，与 `resources.zig` 中路径一致 |

## 如何重新生成

在仓库根目录执行：

```bash
python3 scripts/gen_luna_resources.py
```

生成逻辑与调色板均在脚本内；**不以任何商业系统资源为描摹对象**。授权见同目录 `manifest.json`。

## 目录说明

- `icons/` — 壳层图标
- `taskbar/`、`titlebar/` — 任务栏与标题栏贴片（**唯一权威路径**，不再使用重复的 `ui/` 子树）
- `cursors/`、`wallpapers/` — 光标与壁纸
